Utilities¶

In general, functions listed below will not modify the input objects and return a copy instead, while methods for pypianoroll.Track and pypianoroll.Multitrack may modify the input objects.

Functions¶

pypianoroll.check_pianoroll(arr)[source]¶

Check if the array is a standard pianoroll matrix. Return True if it is. Otherwise, return False. Raise TypeError if the input object is not a numpy array.

pypianoroll.assign_constant(obj, value)[source]¶

Assign a constant value to the nonzeros in the pianoroll(s). If a pianoroll is not binarized, its data type will be preserved. If a pianoroll is binarized, it will be casted to the type of value.

Parameters

value (int or float) – The constant value to be assigned to the nonzeros of the pianoroll(s).

pypianoroll.binarize(obj, threshold=0)[source]¶

Return a copy of the object with binarized pianoroll(s).

Parameters

threshold (int or float) – Threshold to binarize the pianoroll(s). Default to zero.

pypianoroll.clip(obj, lower=0, upper=127)[source]¶

Return a copy of the object with pianoroll(s) clipped by a lower bound and an upper bound specified by lower and upper, respectively.

Parameters

  • lower (int or float) – The lower bound to clip the pianoroll. Default to 0.

  • upper (int or float) – The upper bound to clip the pianoroll. Default to 127.

pypianoroll.copy(obj)[source]¶

Return a copy of the object.

pypianoroll.pad(obj, pad_length)[source]¶

Return a copy of the object with pianoroll padded with zeros at the end along the time axis.

Parameters

pad_length (int) – The length to pad along the time axis with zeros.

pypianoroll.pad_to_multiple(obj, factor)[source]¶

Return a copy of the object with its pianoroll padded with zeros at the end along the time axis with the minimal length that make the length of the resulting pianoroll a multiple of factor.

Parameters

factor (int) – The value which the length of the resulting pianoroll will be a multiple of.

pypianoroll.pad_to_same(obj)[source]¶

Return a copy of the object with shorter pianorolls padded with zeros at the end along the time axis to the length of the pianoroll with the maximal length.

pypianoroll.transpose(obj, semitone)[source]¶

Return a copy of the object with pianoroll(s) transposed by semitones semitones.

Parameters

semitone (int) – Number of semitones to transpose the pianoroll(s).

pypianoroll.trim_trailing_silence(obj)[source]¶

Return a copy of the object with trimmed trailing silence of the pianoroll(s).
